Este arquivo sera usado para registrar problemas encontrados no app antes de aplicar correcoes.

- Anotar o erro aqui assim que for identificado.
- Descrever a melhor solucao recomendada.
- Nao alterar codigo, assets ou APK ate receber comando explicito de correcao.

## Referencia de escopo desejado

Prompt mestre recebido em 2026-05-28:
- App FonoLousa para avaliacao fonoaudiologica infantil.
- Tela inicial com nome obrigatorio da crianca, botao Iniciar Avaliacao e botao Ver Relatorios.
- Fluxo fixo com 3 atividades: Nomeacao de Figuras, Repeticao e Discriminacao Auditiva.
- Cada atividade deve usar uma amostra equilibrada de estimulos; regra atual: 3 estimulos unicos por categoria.
- O terapeuta marca Acertou ou Errou e o app avanca automaticamente.
- Relatorios analiticos com grafico de barras por categoria e grafico de linha por evolucao historica.
- Banco desejado no prompt Flutter: criancas, sessoes e resultados_atividades.
- Regra funcional importante: nao implementar personalizacao de estimulos nem treino por fonemas.
- Interface minimalista e focada no uso por terapeutas.
